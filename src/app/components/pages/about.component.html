About

The Bootstrap Expo is a showcase of beautiful and inspirational sites using Bootstrap, the popular open source front-end framework. It’s designed, built, and maintained by @mdo, co-creator of Bootstrap, to show what’s possible when using Bootstrap.

Submissions

We do not accept all submissions. We only select high quality, beautiful, and comprehensive examples of building with Bootstrap. Sites appearing too similar to Bootstrap’s documentation or default styles will not be accepted. Submissions with inappropriate content will also not be accepted.

How to submit

To submit a site suggestion, open an issue or create a pull request. (Any new pull request should conform to the Code Guide).

About the site

It’s built with Jekyll, developed on GitHub, and is hosted on GitHub Pages.
